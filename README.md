# Asteroid Attack 🚀

Game 2D berbasis fisika bertema luar angkasa. Pemain mengendalikan pesawat untuk **menggiring asteroid** ke area koleksi hijau menggunakan mekanik pantulan fisika.

## 🎮 Cara Main
- Gunakan **WASD** atau **Arrow Keys** untuk menggerakkan pesawat
- Dorong asteroid ke **area hijau** untuk mengumpulkannya
- Tidak ada skor, tidak ada nyawa — hanya pengalaman santai!

## 🛠️ Teknologi
- **Game Engine:** Godot 4.5.1
- **Bahasa:** GDScript
- **Platform:** macOS / Windows
